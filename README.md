## О проекте

![CI GitHubAction](https://github.com/MakcuM-CABUH/games_oop_javafx/blob/main/.github/workflows/maven.yml)

Проект шахматы - консольное приложение. Пользователю предлагается подвигать шахматные фигуры по доске.

Программа может:

1. Отображать положение фигур.

2. Изменять положение фигур на доске в соответствии с тем, какой ход произвёл пользователь.

3. Проверять допустимость выбранного хода для каждого типа фигуры в соотвтетствии с правилами игры в шахматы.

4. Удалять фигуры, "съеденные" в процессе игры.

5. Объявлять победителя в соотвтествии с правилами игры в шахматы.
