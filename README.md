## О проекте

![CI GitHubAction](https://github.com/peterarsentev/job4j_tracker/actions/workflows/maven.yml/badge.svg)
                   https://github.com/MakcuM-CABUH/games_oop_javafx/
Проект шахматы - консольное приложение. Пользователю предлагается подвигать шахматные фигуры по доске.

Программа может:

1. Отображать положение фигур.

1. Изменять положение фигур на доске в соответствии с тем, какой ход произвёл пользователь.

1. Проверять допустимость выбранного хода для каждого типа фигуры в соотвтетствии с правилами игры в шахматы.

1. Удалять фигуры, "съеденные" в процессе игры.

1. Объявлять победителя в соотвтествии с правилами игры в шахматы.
